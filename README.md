# 📝 Notes App

A simple and stylish **Notes App** built with HTML, CSS, and JavaScript. It lets you create, edit, and delete notes right in your browser, with automatic saving using `localStorage`.

## Features

- Create unlimited notes with one click
- Edit notes directly
- Delete notes with a click on the trash icon
- Notes auto-save and persist using localStorage
- Clean, responsive, golden-brown gradient design

## Technologies Used

- HTML
- CSS (Custom design using gradients and filters)
- Vanilla JavaScript
